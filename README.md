# Digital Twin OS

AI-powered digital twin platform for strategic decision-making.

## Structure

- `app/(website)/` - Public website (Home, About, Pricing)
- `app/(os)/` - Digital Twin OS application (Dashboard, Council, Twins, etc.)
- `src/os/` - Backend engines (Twin, Council, Skill, Marketplace, Memory)
- `src/twins/` - Twin intelligence profiles (JSON)
- `src/styles/` - Design tokens and global styles
- `app/api/` - API routes

## Getting Started

```bash
npm install
npm run dev
```
